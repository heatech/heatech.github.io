配置使用Bitnami虚机版本的redmine

发表于 2015-09-06   |   分类于 开发工具   |  

简介

Redmine是用Ruby开发的基于web的项目管理软件，是用ROR框架开发的一套跨平台项目管理系统，据说是源于Basecamp的ror版而来，支持多种数据库，有不少自己独特的功能，例如提供wiki、新闻台等，还可以集成其他版本管理系统和BUG跟踪系统，例如Perforce、SVN、CVS、TD等等。这种 Web 形式的项目管理系统通过“项目（Project）”的形式把成员、任务（问题）、文档、讨论以及各种形式的资源组织在一起，大家参与更新任务、文档等内容来推动项目的进度，同时系统利用时间线索和各种动态的报表形式来自动给成员汇报项目进度。
— 百度百科

bitnami提供了各种开源项目安装包，帮助你解决安装中依赖问题。redmine的一键安装可以从这里下载，页面右侧还提供了Virtual Machines的版本下载。本文所介绍的redmine，就是基于这个版本的配置。

了解环境

在线README文件介绍了这台虚机中已经安装好的bitnami定制版本redmine的基本信息，例如环境、目录组成。

README

比如说在Bitnami Redmine Stack 3.1.0-0中，集成的套件有：

  • Redmine 3.1.0
  • Apache 2.4.16
  • ImageMagick 6.7.5
  • MySQL 5.5.42
  • Subversion 1.8.13
  • Git 1.9.5
  • Ruby 2.0.0-p645
  • Rails 4.2.2
  • RubyGems 1.8.12

这些版本信息很重要，你要知道，开源世界里更新迭代很快，如果版本对不上，特别是大版本，会给系统带来不稳定。这些软件的安装根目录在/opt/bitnami。相应的redmine的安装根目录在/opt/bitnami/apps/redmine/htdocs。通过以下命令可以查看安装好的mysql数据库信息。

设置主题

redminecrm是一家基于redmine销售插件及开发服务的公司。其提供了几套比较不错的redmine主题，并且这些主题目前是免费的。redminecrm主题的下载唯一要求是需要注册并且登录，由于bitnami的虚机没有desktop，通过命令行wget很难完成，所以可以先从windows上进行注册下载，再在虚机里部署。

  1. 在Windows上注册用户，并且下载circle主题

  2. 通过putty的pscp命令复制到bitnami虚机。

  3. 在bitnami虚机里解压主题文件到/opt/bitnami/apps/redmine/htdocs/public/themes目录，对于application.css文件的路径应该类似/opt/bitnami/apps/redmine/htdocs/public/themes/circle/stylesheets/application.css。

  4. 重起apache服务
  5. 进入redmine的配置界面，选择circle主题。
